Password Exposure Analyzer - Requirements

System Requirements
- Windows 10/11, macOS, or Linux
- Node.js 20 or newer
- npm 10 or newer
- PostgreSQL 16 or newer
- Docker Desktop optional, recommended for easiest local setup

Frontend Dependencies
- React
- TypeScript
- Tailwind CSS
- React Router
- Axios
- Recharts
- Lucide React
- Vite

Backend Dependencies
- Node.js
- Express.js
- TypeScript
- PostgreSQL pg driver
- JWT authentication
- bcrypt password hashing
- Helmet security headers
- CORS
- Express Rate Limit
- Zod validation
- Cookie Parser
- CSV Stringify
- dotenv

Database Requirements
- PostgreSQL database named password_exposure_analyzer
- Migration script: backend/db/migrations/001_init.sql
- Seed script: backend/src/db/seed.ts

Environment Variables
- NODE_ENV
- PORT
- CLIENT_ORIGIN
- DATABASE_URL
- JWT_SECRET
- JWT_EXPIRES_IN
- JWT_REFRESH_EXPIRES_IN
- BCRYPT_ROUNDS
- RATE_LIMIT_WINDOW_MS
- RATE_LIMIT_MAX
- ACCOUNT_LOCKOUT_ATTEMPTS
- ACCOUNT_LOCKOUT_MINUTES

Install Commands
Backend:
cd backend
npm.cmd install

Frontend:
cd frontend
npm.cmd install

Run With Docker
cd "C:\CHRIST\IISC\New folder"
copy .env.example .env
docker compose up --build

Run Without Docker
1. Install and start PostgreSQL.
2. Create the password_exposure_analyzer database.
3. Update DATABASE_URL in .env.
4. Run backend:
   cd backend
   npm.cmd run migrate
   npm.cmd run seed
   npm.cmd run dev
5. Run frontend in another terminal:
   cd frontend
   npm.cmd run dev

Local URLs
- Frontend: http://localhost:5173
- Backend API: http://localhost:4000/api
- Health Check: http://localhost:4000/api/health
